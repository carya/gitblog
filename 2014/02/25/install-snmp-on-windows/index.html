Feb 25, 2014 in 系统运维

本文简要记录了在Windows上安装 net-snmp 5.7.1的步骤，最新的源码包可上net-snmp官方网站下载。

安装net-snmp 5.7.1之前需要先安装

  1. VS2010
  2. Win32 OpenSSL v1.0.1f
  3. ActivePerl 5.16.3
